# Cheerlot Android

React Native (Expo) 기반 안드로이드 앱 프로젝트

## 기술 스택

- React Native (Expo)
- JavaScript

## 프로젝트 구조

```
src/
├── screens/      # 화면 컴포넌트
├── components/   # 재사용 UI 컴포넌트
├── hooks/        # 커스텀 훅 (React 상태 로직)
├── services/     # API 호출 함수
├── utils/        # 유틸리티 함수
├── constants/    # 상수 값
└── assets/       # 이미지, 폰트
```

## 코딩 가이드라인

- 클린 코드를 지향한다
- 실무적인 관점에서 코드를 작성한다
- 불필요한 추상화를 피하고 단순하게 유지한다
- 컴포넌트는 단일 책임 원칙을 따른다
- 의미 있는 변수명과 함수명을 사용한다
- 주석보다 코드 자체가 의도를 설명하도록 작성한다
